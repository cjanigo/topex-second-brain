## Usage Notes

- **Fixed Fee vs T&M:** Use Fixed Fee for well-defined desktop studies. Use T&M or NTE for projects with unknowns (permitting timelines, agency feedback rounds).
- **Task 1 (PM) is always included.** Copy the boilerplate language verbatim.
- **Task 4 (Reimbursable) is always included** unless project is fully remote/desktop with no travel.
- **"Engineering Services" on invoices:** Standard language — do not itemize by task on invoices.
- **Multi-agency coordination:** If City/County review is required, call it out explicitly in the task that triggers it. Avoids scope disputes if coordination takes extra rounds.
- **30% cost estimates:** Appropriate for feasibility/alternatives studies. Note the +/- to manage expectations.
- **Desktop-only studies:** Explicitly state data sources used (DOGAMI LIDAR, county GIS, etc.) to set expectations on accuracy.
- **Attachment A:** Always attach General Terms and Conditions. Reference it in the body as "Attachment A."
