## Usage Notes

- **50% down / 50% on delivery** is standard for survey work. Adjust only if client is a repeat account or municipality.
- **Credit card fee:** Add 3% line item if client will pay by card. Omit if check/ACH.
- **Task numbering:** Keep it simple (Task 1 = field, Task 2 = office). Add Task 3 if there is a recording or filing step.
- **Deliverable specificity:** Name the exact outputs (number of legal descriptions, sheet size, format). Avoids scope creep disputes.
- **Exclusions:** Always include the boilerplate exclusion list. Add project-specific exclusions as needed.
- **Governing law:** Oregon by default for Lincoln County and most Oregon projects.
