## Scope Summary

Baseline scope per CWES Task Order #1 (MSA executed 2026-03-03). Used for scope change comparison.

## Scope of Work

Per CWES Engineering Scope of Services (Attachment A):

**Task 1 - Project Management:** Coordination with the District as needed.

**Task 2 - Mapping:**
- Collect, scan, and digitize existing water system maps and records into GIS
- Create GIS base map for both OWD (~560 connections) and Cape Meares (~236 connections) systems
- Map features: tank locations, elevations, pump stations, pipelines, valves, hydrants

**Task 3 - Modeling (EPANET):**
- Build computer water model from GIS map
- Calibrate with real-time pressure data and fire hydrant test results
- Enable scenario modeling for system improvements

**Not Included:** Field GPS locating of system features, field surveying
